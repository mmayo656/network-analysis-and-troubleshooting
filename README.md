# network-analysis-and-troubleshooting
This project demonstrates basic network analysis and troubleshooting skills using Wireshark and Cisco Packet Tracer.

---

## Skills Demonstrated

- Packet capture analysis using Wireshark
- Identifying protocols (TCP, UDP, HTTP, DNS)
- Understanding IP addressing and network communication
- Basic network design using Cisco Packet Tracer
- Troubleshooting connectivity issues

---

## Wireshark Analysis

Captured and analyzed network traffic to identify:

- DNS queries and responses
- TCP handshake process
- HTTP request and response flow
- Source and destination IP addresses
- Port usage

### Example Findings

- Identified TCP 3-way handshake (SYN, SYN-ACK, ACK)
- Observed DNS resolution process
- Analyzed HTTP traffic between client and server

---

## Packet Tracer Network Design

Built a basic network topology including:

- Router
- Switch
- Multiple end devices

Configured:

- IP addressing
- Network connectivity
- Device communication

## Real-World Application

These skills apply to:

- Troubleshooting network connectivity issues
- Identifying network traffic problems
- Understanding how systems communicate across a network
- Supporting IT infrastructure environments
